# Features

The list below reflects the [BetterSEQTA+ README](https://github.com/BetterSEQTA/BetterSEQTA-Plus/blob/main/README.md). Exact UI labels may change between releases.

## Appearance and layout

- **Dark mode** and improved styling for SEQTA Learn.
- **Custom backgrounds and themes**, including a theme store and fully customizable themes.
- Options to **hide or remove certain side menu items** you do not need.

## Home and daily workflow

A **custom home page** can include:

- Daily lessons
- Shortcuts
- Easier access to notices
- Assessments

## Grades and schedule

- **Grades calculator** to help you work with assessment data in context.
- **Next-lesson notification**, shown about five minutes before the current lesson ends (timing depends on timetable data from SEQTA).

## Platform support

- Chromium-based browsers are the primary target.
- Firefox is supported via Mozilla Add-ons.
- Safari support is experimental; see the main repository for limitations.

For theme authoring beyond built-in options, see [Customization](customization.md).
