# BetterSEQTA+

**BetterSEQTA+** is an open source browser extension that adds features and improves the experience of [SEQTA Learn](https://www.seqta.com.au/seqta-learn) for students and staff.

For downloads, release notes, and the project homepage, see **[betterseqta.org](https://betterseqta.org/)**. Source code and issues live on **[GitHub](https://github.com/BetterSEQTA/BetterSEQTA-Plus)**.

## What you will find here

This site covers installation, features, customization (including themes and the full theme creation reference), plugins, architecture, contributing to the extension, and troubleshooting. Content is aligned with the **[BetterSEQTA+ repository](https://github.com/BetterSEQTA/BetterSEQTA-Plus)** on GitHub. It is built with **[Zensical](https://zensical.org/)** (Markdown → static HTML).
